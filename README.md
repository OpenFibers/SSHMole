#SSHMole

OSX 下使用 ssh 穿越功夫网。  

##使用说明
没那么多乱七八糟的规则，填上 ssh 帐号就开启穿越之旅。目前仅支持全局代理。  
Master分支开发中，有很多bug，请下载tag上的0.1或0.2版本使用。
